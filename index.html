Setelah mengikuti Coding Mum saya berkesempatan untuk mengikuti project dari Garuda, yaitu pembuatan kios digital. Selain itu saya juga membuat website sendiri yaitu catatancoding.com. Selama belajar di kelas Coding Mum, pengalamanya seru dan teman-teman baru dibantu oleh mentor2 yg profesional. About

KollaSpace hadir untuk ikut mengenalkan literasi teknologi agar masyarakat tidak hanya menjadi pengguna teknologi saja, namun juga memahami, mengembangkan dan mampu membuat teknologi sebagai solusi masalah sehari-hari. KollaSpace memiliki moto, "Lembaga edukasi untuk literasi teknologi".
